# Fake News Detection Web Application

An AI-powered web application that detects whether a news article is Fake or Real using Natural Language Processing and Machine Learning.

# Features
- Classifies news articles as Fake or Real
- Displays prediction confidence score
- Interactive and user-friendly web interface
- High accuracy using machine learning

# Technologies Used
- Python
- Natural Language Processing (NLP)
- TF-IDF Vectorization
- Logistic Regression
- Streamlit
- Pandas, Scikit-learn, NLTK

# Model Performance
- Achieved 98.5% accuracy on the test dataset
- Trained on a clean and balanced fake news dataset
